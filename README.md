# 📚 Library Management System

A comprehensive Java-based Library Management System that provides an interactive command-line interface for managing books and library operations.

## ✨ Features

- 📖 **Display Books**: View books by category
- 📚 **Borrow Books**: Check out books from the library
- ➕ **Add Books**: Add new books to the collection
- 🔍 **Search Books**: Find books in the library
- 📝 **Request Books**: Request new books to be added
- 📋 **Check Requests**: View all pending book requests
- ❌ **Remove Books**: Remove books from the system

## 📚 Book Categories

The system includes the following default categories:
- Computer Science
- Biology
- Mathematics
- Chemistry
- Physics
- English
- Others

## 🛠️ Technical Features

- Robust input validation using Try-Catch blocks
- Array bounds checking with While loops
- Dynamic book management using ArrayLists
- Category-wise organization of books and authors
- User-friendly command-line interface
- Seamless navigation between operations

## 🚀 Getting Started

### Prerequisites
- Java Development Kit (JDK) installed
- Any Java IDE or command line interface

### Running the Program
1. Clone the repository
2. Navigate to the project directory
3. Compile the Java file:
   ```bash
   javac src/project/Project.java
   ```
4. Run the program:
   ```bash
   java -cp src project.Project
   ```
